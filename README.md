# engined-member

Member system service for engined, which contains member management, permission mechanism, Restful APIs and HTTP middlewares.

## Installation

Using NPM to install module:

```shell
npm install engined-member --save
```

## Dependencies

* engined-validator
* engined-storage
* engined-http
* engined-mailer
* engined-mysql`
